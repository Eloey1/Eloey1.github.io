GAME PROJECT

Pawn's Gambit

C# • Unity • 2025 • The Rookies Runner Up

About the Project

Pawn's Gambit is a puzzle game built in Unity. My role was primarily gameplay programming and tool development for the level designers.

The core mechanic revolves around chess movement rules applied to a dungeon crawler setting. I implemented a custom A* pathfinding system to handle the unique movement constraints.
